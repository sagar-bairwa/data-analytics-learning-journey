# 🎓 Student Depression Analysis Dashboard

An end-to-end data analytics project exploring how academic pressure, financial stress, sleep patterns, and study habits relate to student well-being.

The project combines SQL-based data preparation with Tableau dashboard development to transform raw student data into meaningful insights.

## 📌 Project Objective

The objective of this project was to analyze factors that may influence student depression and identify patterns within academic, financial, and lifestyle-related variables.

Key questions explored:

- How does academic pressure vary among students?
- Does financial stress impact student well-being?
- What relationship exists between sleep duration and mental health?
- How does study satisfaction differ across students?
- What patterns can be observed in study hours?
